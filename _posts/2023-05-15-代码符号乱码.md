* 代码符号乱码

找到仓库目录

``` cmd
\node_modules\hexo-prism-plugin\src\index.js
```

在变量中添加大括号的映射，e.g.

``` js
const map = {
  '&#39;': '\'',
  '&amp;': '&',
  '&gt;': '>',
  '&lt;': '<',
  '&quot;': '"',
  '&#123;': '{',
  '&#125;': '}'
};
```
